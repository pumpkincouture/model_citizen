# ModelCitizen

Provides validation methods to LumberYard Sinatra app and LumberYard command line app models.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'model_citizen'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install model_citizen
